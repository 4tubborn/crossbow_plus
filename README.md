# Crossbow Plus
A Minecraft datapack that modifies the logic of crossbows.

* supported versions: 1.21.5-1.21.10
* develop in 1.21.11

## Features
### More Projectiles
* All blocks and items can be charged into crossbows!

### More Special Projectiles
* Tnt, shulker bullets, wind charges, fireballs, end pearls, potions, spawn eggs and more! They can be shot as special entities, what a great deal of fun!

### Bundles
* Now you can charge projectiles from a bundle!

### Compatible
* Projectiles are compatible with inifinity, multishot enchantments!

### New enchantments
* Add damage - add arrow damage.
* Reduce damage - reduce arrow damage.
* Headwear - blocks will be worn over the head.
* Instataneous explosion - tnt will explode immediately.
